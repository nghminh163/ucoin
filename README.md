# Prerequisites

- Docker
- Docker Compose

# How to run

1. Clone this repo

```bash
git clone --recurse-submodules https://github.com/nghminh163/ucoin
cd ucoin
```
2. Start the docker containers

```bash
docker-compose up -d --build
```

3. Access the UI to test the blockchain at [http://localhost:3001](http://localhost:3001)
